Safety instructions

1. The operator is required to be familiar with Aubo - i5 robot 's operation manual, regulate the operation process.

2. Program is completed after the completion of the robot simulation operation, to ensure correct after the actual operation.

3. In case of unexpected situations, please quickly press the emergency button to ensure safety.

4. Inspector PIM60 before using reference objects, coordinate system and other settings, see Inspector PIM60 operating manual.

I. Overview：

      The Camera plug-in can obtain the position information of the object through the Inspector PIM60 and feedback the position information to the manipulator. The feedback information is used by the manipulator to assist positioning, which greatly increases the flexibility of the manipulator in the practical application.

III. Setting instructions

      1. Select setting -> peripheral -> Camera interface.

      2. Set the IP address of the Inspector PIM60 in the Settings tab, for example 192.168.0.101; select the save path for the snapshot, for example / root / SICK_Pictures.

Figure 5 Inspector PIM60 initialization settings

      3. Set Coordinate Compensation, which is the total compensation each time the Inspector PIM60 passes to the robot arm in meters. After setting, click the Save button.

      4. Click the init Camera button, the camera will take pictures, you can view the picture in the Camera tab, the specific content in the tab has a detailed description.

V. user interface

      User related settings, you can use the relevant interface in the script, as follows:

      get_waypoint_from_camera(

          const double &compensationX,

          const double &compensationY,

          double &positionZ,true)

      The first two parameters are camera coordinates. The third parameter is the height of the Z direction to which the manipulator moves. The last parameter is to determine if the Inspector PIM60 takes pictures in meters. Normal completion After the function is finished, the six joint angles of the manipulator will be returned, in radians.

      move_joint({j1,j2,j3,j4,j5,j6},true)

      Incoming parameters are the six joint angles of the manipulator, in radians, moving to the position within the joint space.

      move_line({j1,j2,j3,j4,j5,j6},true)

      Incoming parameters are the six joint angles of the manipulator, in radians. Move linearly to position in space.
